# DBMS Quora (quora_posts_dbms)

A full-stack Quora-like application for DBMS questions, built with React, Node.js, Express, and MySQL.

## Features
- Ask and Answer DBMS questions
- Authentication (Signup/Login)
- Global Session Handling (Axios Interceptors)
- AI Voice Help integration (VAPI)

## Tech Stack
- **Frontend**: React + Vite, TypeScript, Tailwind CSS, Lucide Icons
- **Backend**: Node.js, Express
- **Database**: MySQL

## Setup
1. Clone the repository
2. Install dependencies: `npm install` in both `client` and `server`
3. Set up environment variables in `server/.env` (refer to `.env.example`)
4. Run the app: `npm run dev` in both directories
